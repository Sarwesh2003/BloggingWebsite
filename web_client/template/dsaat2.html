Automata Theory Limitations and Applications

The term "automata" comes from the Greek "αὐτόματα"
meaning "self-acting". An automaton (plural of Automata)
is an abstract, self-propelled computing device
that automatically performs a given set of tasks.

Applications of Automata

Lexical Analyser

  1. Lexical analysis is the first step in the compiler, also known as a scanner. Converts a high-level input program into a sequence of tokens.
  2. Lexical analysis can be implemented using deterministic finite automata.
  3. The result is a sequence of tokens sent to the parser for parsing

For designing the parsing phase of a compiler (Syntax Analysis).

Parsing is used to obtain a string using grammar generation rules. Used to check if a string is acceptable. The compiler is used to check the correct syntax of a string. The parser takes input and builds a parse tree.
A parser can be of two types:

  • Top-Down Parser: Top-down parsing starts from the top with the start-symbol and derives a string using a parse tree.
  • Bottom-Up Parser: Bottom-up parsing starts from the bottom with the string and comes to the start symbol using a parse tree.

Limitations of Automata

  1. Any non-deterministic Finite automata can be converted to a corresponding deterministic Finite automata, but not every non-deterministic PDA can be converted to a deterministic PDA.
  2. There is no state machine that can find and recognize the same set of binary strings of zeros and ones.
  3. The expressive power of non-deterministic PDAs is greater than that of deterministic PDAs. This means that there are contextless grammars from which you cannot build a deterministic PDA. For example, you cannot build a deterministic PDA for L={w|w is a palindrome}.
  4. The algorithm for determining whether there is a deterministic PDA for a grammar is unknown.
  5. PDA can not apprehend context-touchy and unrestricted grammar.
  6. Unable to determine whether PDA recognizes all sets of strings for input characters

Conclusion

Automata theory is one of the major branches of computational theory concerning how effectively algorithms can be used to solve problems in computational models. The four automata suites are capable of accurately performing tasks such as lexical analysis, speech recognition and more. Of course there are other automaton applications too, but by far the most popular is above. But it also has certain limitations as none of the four automata families have been able to eliminate its shortcomings.
