Discrete Structures and Automata Theory

Automata Theory Limitations and Applications

The term "automata" comes from the Greek "αὐτόματα"
meaning "self-acting". An automaton (plural of Automata)
is an abstract, self-propelled computing device
that automatically performs a given set of tasks.

Introduction

Automata theory is essentially about studying various mechanisms for producing and recognizing language. Automata theory is basically useful for studying different types of grammar and automata. Grammar is a mechanism for generating sentences in a language. Automata are language recognition mechanisms. Automata theory is mainly used to study the relationship between various types of automata as speech recognition devices and their grammar.

Automata theory deals with the definitions and properties of mathematical computational models.

  • Finite-state machines are used in word processing, compilers, and hardware design.
  • Context-free grammar is used in programming languages and artificial intelligence.
  • Turing machines represent computable functions.

Families of Automata

There are four major families of automaton :

  1. Finite-state machine.
  2. Pushdown automata.
  3. Turing machine.
  4. Linear-bounded automata.

Finite-State Machine (FSM)

Finite-state machines are an ideal computational model for small amounts of memory and do not maintain memory. This mathematical model of the machine can only reach a finite number of states and transitions between these states. Its main use is in mathematical problem analysis. Finite-state machines are also used for purposes other than general computation, such as recognizing regular languages.

Pushdown Automata

Pushdown automata allow context-free languages with a set of plain languages. A language that describes a string with matching parentheses is a language without a context. Let's say a programmer has written some code, and all parentheses must match for the code to be valid. One way to do this is to pass code (in string form) to a down-push automaton that is programmed with a jump function that implements a context-free grammar for the balanced bracket language.

Linear-bounded Automata

A linear bounded automaton (lba) is essentially a Turing machine whose computation is limited by the amount of tape to which the input is written. You can think of it as consisting of a finite set of states, a finite alphabet (with special right and left end markers [ and ]), a specified initial state, and a finite set of instructions in the form of a Turing's quad. machine.

Turing Machine

A Turing Machine(TM) is a mathematical model consisting of an infinite length tape divided into cells that receive input data. It consists of a head that reads the input tape. The status register stores the state of the Turing machine. After reading the entered character, it is replaced by another character, its internal state is changed, and it moves right or left in one cell. When the TM reaches its final state, the input string is accepted, otherwise it is rejected.
